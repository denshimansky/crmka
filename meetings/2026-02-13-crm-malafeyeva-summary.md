## Ключевые темы обсуждения

1. **AI-инструмент для анализа CRM** — Денис научил AI работать в браузере совместно (видеть действия пользователя). Планирует использовать при анализе Альфа-CRM на следующей неделе, чтобы избежать двойной работы (смотреть → пересказывать).

2. **Планирование кастдевов (CustDev)** — определение пула респондентов для интервью: внутренние сотрудники + клиенты CRM + пользователи чужих CRM.

3. **Распределение ролей для внутренних кастдевов** — составлен список сотрудников по ролям:
   - **Наташа** — финансы, расходы (роль: финансист)
   - **Оля** — посещения, внесение данных (роль: старший админ). Временно не привлекать (декрет, не в ресурсе)
   - **Настя С.** — рутина, обычный админ (низшая роль)
   - **Настя Т.** — ещё один админ
   - **Яна** — новый админ (полгода), может дать свежий взгляд

4. **Выбор клиентов для кастдевов** — две категории:
   - *Активные пользователи функционала:* Фирма (максимальный функционал, отчётность), Чернова (может рассказать про админскую роль «снизу»), Суслова, Козловских (разбирается в цифрах)
   - *Самопришедшие клиенты (без прямой продажи):* Семенюк (~год пользуется, пришла из Instagram, сама разобралась), Харламова (тоже из Instagram), Иванова — интересны тем, что выбрали сложную CRM сами, без уговоров. Понимание их мотивации = ключ к маркетингу без личных продаж
   - *Антипод:* Горбунова — крупный клиент, но использует только стартовую страницу (нужен только дашборд)

5. **Пользователи чужих CRM** — Темникова на Альфа-CRM (более адекватна в цифрах), Маша (первая CRM). Денис попросил время подумать, каких ещё внешних респондентов искать.

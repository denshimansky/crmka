### Инфраструктура

15. ~~**VPS-провайдер**~~
    - **Статус:** решён, 25.03.2026
    - **Решение:** Timeweb Cloud. Поэтапно: MVP — 1 VPS (СПб/Мск, ~3-5к₽/мес), 500 партнёров — PostgreSQL 3-node HA + 2 app-сервера + балансировщик (~15-21к₽/мес), мировая арена — гео-распределение через дата-центры TW в Европе (Амстердам/Франкфурт).

16. ~~**Домен для API**~~
    - **Статус:** решён, 25.03.2026
    - **Решение:** app.umnayacrm.ru. Отдельный API-домен не нужен — Next.js фулстек, API routes на /api/*.

17. ~~**Бэкапы**~~
    - **Статус:** решён, 25.03.2026
    - **Решение:** pg_dump ежедневно в 3:00, хранение 30 дней в Timeweb S3 (~639₽/мес). WAL-архивирование для point-in-time recovery. Тест восстановления раз в месяц. На этапе 3 — streaming replication в европейский ДЦ Timeweb через WireGuard VPN.

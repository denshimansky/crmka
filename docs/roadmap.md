### Доработки v1.5.0–v1.5.2 (после аудита)

42+ фичи добавлены за 2 дня (09–10.04.2026):

**CRM:**
- CRM-02: История коммуникации — модель Communication, лента в карточке
- CRM-07/ADM-08: Справочник каналов привлечения — CRUD, страница /settings/channels
- CRM-10: Дубликаты — страница /crm/duplicates с поиском и предупреждениями
- CRM-17: Отчёт «Допродажи и возвраты» — /reports/crm/upsell
- CRM-22: Быстрое создание лида — кнопка «+» на дашборде и /crm/leads

**Расписание:**
- SCH-07: Перевод между группами — API /api/enrollments/[id]/transfer
- SCH-08: Замена инструктора — substituteInstructorId в Lesson

**Посещения:**
- ATT-10: Отчёт «Потенциальный отток» — /reports/churn/potential

**Финансы:**
- FIN-26: Drill-down — выдвижная панель с деталями в P&L, выручке, ДДС
- FIN-27: Экспорт — компонент ExportButton + ReportExport (Excel)

**Администрирование:**
- ADM-05: Wizard онбординга — 6 шагов (563 строки)
- ADM-09: Причины отчисления — справочник
- ADM-09a: Причины пропусков — /settings/absence-reasons
- ADM-11: Импорт клиентов — /crm/import (CSV/XLSX)
- ADM-03: Настройка прав ролей — /settings/role-permissions (матрица 261 строка)
- PageHelp: справка «?» на всех страницах (1112 строк контента)

**Бэк-офис:**
- Impersonation: «Войти как партнёр» с аудитом
- Управленческий дашборд: MRR, партнёры, спящие, топ-10

**Инфраструктура:**
- Супертест: 3 филиала, 50 клиентов, 150 абонементов, 1126 занятий
- Троттлинг API: 80 req/min + retry на 429
- Шаблоны скидок: /settings/discount-templates

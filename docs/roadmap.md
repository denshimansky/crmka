### Реализовано в v1.5.0–v1.5.3 (после аудита 08.04)

60+ фич добавлены за 3 дня (09–11.04.2026):

**CRM:**
- CRM-02: История коммуникации — модель Communication, лента в карточке
- CRM-05: Допродажи/возвраты — clientStatus, UI в карточке клиента
- CRM-07/ADM-08: Справочник каналов привлечения — CRUD, /settings/channels
- CRM-10: Дубликаты — /crm/duplicates с поиском и предупреждениями
- CRM-12: Объединение дубликатов — полный merge API (190 строк)
- CRM-17: Отчёт «Допродажи и возвраты» — /reports/crm/upsell
- CRM-18: Автосортировка лидов — по nextContactDate (дефолт)
- CRM-22: Быстрое создание лида — кнопка «+» на дашборде и /crm/leads

**Расписание:**
- SCH-03a: Архивирование группы — кнопка в карточке группы, soft delete
- SCH-04: Фильтры по кабинетам/инструкторам/направлениям — ScheduleFilterableGrid
- SCH-05: Цветовая индикация заполняемости — 3 уровня (зелёный/жёлтый/красный)
- SCH-07: Перевод между группами — API /api/enrollments/[id]/transfer
- SCH-08: Замена инструктора — substituteInstructorId + ЗП заменяющему
- SCH-09: Массовая отмена занятий — CancelDayDialog (праздники, карантин)
- SCH-11: Разовые занятия/отработки — isMakeup + API + UI

**Посещения:**
- ATT-09: Отчёт «Неотмеченные дети» — /reports/attendance/unmarked
- ATT-10: Отчёт «Потенциальный отток» — /reports/churn/potential (threshold 3+)

**Абонементы:**
- SUB-07: Связанная скидка — автопересчёт при отчислении (recalcLinkedDiscounts)
- SUB-11: Возврат абонемента — полный flow (preview + refund + деактивация)

**Финансы:**
- FIN-15: P&L формат B — отдельный /api/reports/pnl-by-direction
- FIN-16: Автораспределение расходов — expense-distribution.ts (141 строка)
- FIN-18: Возврат средств — /api/payments/refund (транзакция + аудит)
- FIN-21: Онлайн-оплата webhook — ЮKassa (idempotency, IP whitelist, Basic Auth)
- FIN-26: Drill-down — выдвижная панель с деталями в P&L, выручке, ДДС
- FIN-27: Экспорт — ExportButton + ReportExport (Excel)

**Зарплата:**
- SAL-10: Оплата пробных — payForTrialLessons настройка организации
- SAL-11b: Корректировки прошлых периодов — salary-corrections.tsx + API

**Администрирование:**
- ADM-03: Настройка прав ролей — /settings/role-permissions (матрица 261 строка)
- ADM-05: Wizard онбординга — 6 шагов (563 строки)
- ADM-09: Причины отчисления — справочник
- ADM-09a: Причины пропусков — /settings/absence-reasons
- ADM-11: Импорт клиентов — /crm/import (CSV/XLSX)
- PageHelp: справка «?» на всех страницах (1112 строк контента)

**Бэк-офис:**
- Impersonation: «Войти как партнёр» с аудитом
- Управленческий дашборд: MRR, партнёры, спящие, топ-10

**Инфраструктура:**
- Супертест: 3 филиала, 50 клиентов, 150 абонементов, 1126 занятий
- Троттлинг API: 80 req/min + retry на 429
- Шаблоны скидок: /settings/discount-templates
- Хлебные крошки: auto-breadcrumbs.tsx на всех страницах
- PWA: service worker + manifest + sw-register

**Абонементы:**

- SUB-11: Возврат абонемента — полный flow (preview + refund + деактивация)
- SUB-12: Авто-предложение переноса баланса при создании нового абонемента

**Финансы:**

- FIN-26: Drill-down — выдвижная панель с деталями в P&L, выручке, ДДС
- FIN-27: Экспорт — ExportButton + ReportExport (Excel)

**Зарплата:**

- SAL-11b: Корректировки прошлых периодов — salary-corrections.tsx + API

**Администрирование:**
- ADM-03: Настройка прав ролей — /settings/role-permissions (матрица 261 строка)
- ADM-05: Wizard онбординга — 6 шагов (563 строки)
- ADM-09: Причины отчисления — справочник
- ADM-09a: Причины пропусков — /settings/absence-reasons
- ADM-11: Импорт клиентов — /crm/import (CSV/XLSX)
- PageHelp: справка «?» на всех страницах (1112 строк контента)

**Бэк-офис:**
- Impersonation: «Войти как партнёр» с аудитом
- Управленческий дашборд: MRR, партнёры, спящие, топ-10

**Обзвон:**
- CALL-04: Результаты обзвона → автозапись в Communication историю клиента

**Склад (INV-01..06):**
- INV-01: Закупка товаров на склад — StockItem, StockMovement (purchase), StockBalance
- INV-02: Перемещение склад → кабинет — transfer_to_room, RoomBalance
- INV-03: Баланс кабинета + кнопка «Списать» — /stock/rooms
- INV-04: Амортизация при закупке — amortizationMonths
- INV-06: Отчёт остатки — /stock (по филиалам)
- 3 страницы UI: /stock, /stock/rooms, /stock/movements
- 5 API: stock-items, stock-movements, stock-balances, room-balances

**Кандидаты (CAND-01..04):**
- CAND-01: Список кандидатов — /staff/candidates (фильтр по статусу)
- CAND-02: Карточка кандидата — /staff/candidates/[id]
- CAND-03: Статусы: NEW → INTERVIEW → TRIAL_DAY → HIRED / REJECTED
- CAND-04: История встреч с хронологической лентой
- Перевод в сотрудники: логин, пароль, роль, филиал

**Инфраструктура:**
- Супертест: 3 филиала, 50 клиентов, 150 абонементов, 1126 занятий
- Троттлинг API: 80 req/min + retry на 429
- Шаблоны скидок: /settings/discount-templates
- Хлебные крошки: auto-breadcrumbs.tsx на всех страницах
- PWA: service worker + manifest + sw-register
